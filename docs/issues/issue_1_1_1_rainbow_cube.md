# Issue 1.1.1 - Rainbow Cube

## Criteria
- 3-dimensional cube spinning on an arbitrary axis
- Realistic first-person camera perspective
- Rainbow colours on each face with edges clearly defined
- Stable and clear lighting effects with a moveable light source

## Key Tasks
- [X] Render elementary squares to a window
- [ ] Render a 3D rainbow cube in front of the camera
- [ ] Apply Blinn-Phong (or equivalent) lighting to the scene
- [ ] Animate the cube with rotation and/or movement
